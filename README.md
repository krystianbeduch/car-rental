# Car Rental
A project carried out in high school at web programming class. The project was intended to summarize the learning of website programming.

## General info
The project is a simple implementation of a car rental website. 4 main subpages: home page, contact, order form and a wide range of cars ofert divided into car brands. The website has been extended to be responsive

## Technologies
Project was focusing only on the aspects of **_HTML 5_** and **_CSS 3_** technologies.
