# CarRental
#### A project carried out in high school at web programming class. The project was intended to summarize the learning of website programming, focusing only on the aspects of HTML 5 and CSS 3 technologies.

## General info
##### The project is a simple implementation of a car rental website. 4 main subpages: home page, contact, order form and a wide range of cars ofert divided into car brands.

## Technologies
##### Project was focusing only on the aspects of HTML 5 and CSS 3 technologies.
